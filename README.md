# 머신러닝, 눈으로 보기

정답이 달린 예시에서 기계가 **규칙을 스스로 배운다**는 게 무슨 뜻인지를, 수식·코드 없이 **움직이는 그림과 인터랙션**으로 익히는 비영리 학습 자료입니다. 0번 입문 + 16챕터. (지도학습 중심 — 신경망·딥러닝은 별도 트랙)

> 라이브러리 사용법이 아니라 **멘탈 모델**이에요 — 선을 맞추고(회귀), 경계를 긋고(분류), 과적합에 빠지고, 그걸 막는 과정을 직접 슬라이더로 만지며 봐요.

## 어떻게 보나

별도 빌드·설치가 없습니다. `index.html`을 브라우저로 열면 됩니다. 모든 잔차·결정 경계·오차 곡선·과녁·혼동행렬·ROC·교차검증은 브라우저에서 실시간으로 계산·렌더링됩니다.

## 구조

처음이라면 **0번부터 순서대로**. 척추는 **무엇을 배우나(A) → 일반화(B) → 모델 동물원(C) → 평가(D) → 현실·종합(E)**. 두 기둥은 **일반화(B)** 와 **모델 동물원(C)**.

- **0번 — 데이터에서 규칙을 배운다**: 규칙 짜기 vs 배우기 · 회귀 vs 분류 · 용어 · 코스맵
- **PART A · 무엇을 배우나**
  - 01 지도학습이란 (특징·라벨 · 예시→모델→예측)
  - 02 회귀 — 선을 맞추다 (잔차 · 직접 맞추기 · 자동 내리막)
  - 03 분류 — 경계를 긋다 (직선/곡선 경계 · 확률 · 임계값 · 로지스틱 이름 해명)
- **PART B · 일반화** (기둥①)
  - 04 학습 vs 암기 (훈련/시험 분리 · 누수 · 검증셋 · 하이퍼파라미터)
  - 05 과적합 vs 과소적합 (복잡도 손잡이 · 두 오차 곡선 · 데이터↑)
  - 06 편향-분산 (과녁 · 다시 학습 시 흔들림 · 분해 · 노이즈)
  - 07 규제 (큰 가중치에 벌점 · λ 손잡이 · 적당한 세기)
- **PART C · 모델 동물원** (기둥②)
  - 08 kNN — 이웃에게 묻기 (다수결 · k 손잡이 · 스케일 함정)
  - 09 결정트리 — 스무고개 (질문 사슬 · 계단 경계 · 깊이)
  - 10 앙상블 — 여럿이 모이면 (다수결 · 배깅 · 부스팅)
- **PART D · 제대로 평가하기**
  - 11 정확도의 함정 (불균형 · 혼동행렬 · 네 칸)
  - 12 정밀도와 재현율 (두 질문 · 임계값 줄다리기 · ROC)
  - 13 교차검증 (한 번의 운빨 · k-폴드 · 누수 조심)
- **PART E · 현실과 종합**
  - 14 특징이 절반 (입력이 절반 · 스케일링 · 특징 공학)
  - 15 한 흐름 종합 (데이터→특징→분리→학습→평가→예측 · 함정 지도)
  - 16 너머 · 회고 (SVM 최대 여백 · 비지도/군집 · 딥러닝 다리 · 전체 정리)

## 기술

- 순수 정적 HTML + JavaScript (의존성·빌드 없음)
- 공통 엔진 `shared/lib.js`(VZ 코어 + 스텝 플레이어 VZ.AL + heatColor) + `shared/lib-ml.js`(머신러닝 도메인 엔진 VZ.ML: 좌표 매핑·산점도/적합선/잔차·결정 경계 채색·적합 곡선·train/test 오차 곡선·편향-분산 과녁·결정트리·혼동행렬·정밀도/재현율 막대·ROC·k-폴드 띠) — 모두 순수 함수(상태 → SVG)
- 다크 테마 `shared/theme.css`
